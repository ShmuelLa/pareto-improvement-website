Pareto Improvement Algorithm

This algorithm is used to pareto improve a fractional allocation in order to improve the total algorithms results and complexity.


Usage:

First you will be prompted to enter the number of desired agents and objects
Afterwards you will need to enter the item allocations for each and every agent and item, please note that each item can be allocated to a total sum of 1.0
After entering the desired allocation you will need to enter valuation values for each agent and ibjects. In this algorithm valuations can be positive or negative.
